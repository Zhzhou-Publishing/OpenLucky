# 高光阴影切除（CLI 功能）

## 编排命令

openlucky.py curve levels

参数如下：

- --shadows, -s 阴影裁切千分之几
- --highlights, -h 高光裁切千分之几
- --channel, -c 作用通道 [all|red|green|blue]
- --mode, -m 模式 [clip-only|clip-strech]
- --input, -i 输入文件
- --output, -o 输出文件

## 功能

1. 裁切亮部 shadows 千分之几的部分，从最暗部分开始往亮部走，将其设置为0。输入整数值，比如5，表示千分之五。

2. 裁切暗部 highlights 千分之几的部分，从最亮部分开始往暗走，将其设置为255。输入整数值，比如5，表示千分之五。

3. mode 是 clip-strech 将中间剩余的部分的色值做拉伸，如果是 clip-only 则不做拉伸直接出片。默认模式是 clip-strech。

4. input output 指定输入输出文件。输入文件支持 jpeg、png、bmp 等常见位图格式（IMAGE_EXTENSIONS 这个 list 里有做枚举），也支持至少 8bit 最多 16bit 色深的 tif/tiff，也支持各类 RAW 格式。处理 RAW 格式要参考现有代码。RAW 格式输出后缀加 .tiff，将其输出为 TIFF。
