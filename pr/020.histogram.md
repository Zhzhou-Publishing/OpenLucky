# 输出图片直方图

## 新增命令

openlucky tool histogram

### 命令参数

-i, --input 输入图片文件，支持 image_formats.py 里面的所有 tiff、RAW图和各类位图。

-t, --type 直方图模式类型，可选参数，默认 rgbl，暂定下面的值：

    rgbl，输出三通道直方图数据和亮度数据。亮度数据输出按照 ITU-R BT.601 标准转为单通道灰度做加权平均的直方图数据

-g, --gamma 套用 gamma 值，可选参数，默认1，建议使用 RAW 格式图片的时候使用 2.2 作为 gamma。

-m, --mode 归一化模式，可选参数，默认值为 log，暂定下面的值：

    log，对数归一化
    linear，线性归一化

-n, --normalization 数据归一化，可选参数，留空表示不做归一化，按照实际像素点数量实际展示。有效值为数字，推荐填写为 256。

-d --downsampling 降采样，可选参数，留空表示不做降采样，按照实际色深来输出。有效值为数字，最小值是256，最大值是65536，数值必须是2的指数，推荐填写为 256。

### 输出数据

1. 是一个二级数组。
2. 第一级的数量按照 type 来定，比如 rgbl 这个模式的第一级就是 4 个，分别代表 R, G, B 三个通道的数据和亮度数据。
3. 第二级可容纳最多 65536 个数字（兼容到 16bit 色深的图片）。
4. 如果做了降采样，按照降采样的数据输出对应的个数，如 -d 256 则第二级会输出 256 个数字。

### 处理细节

1. 如果有 gamma 值的设置，请套用 gamma 变换过后再计算直方图。
2. 使用 numpy 计算直方图即可。
3. 数据归一化支持对数归一化和线性归一化。

### 归一化和降采样的参考代码（来自于 gemini）

下面是参考的代码。

```python
import numpy as np

def normalize_linear(hist_data, normalization=256):
    # hist_data: 长度为 65536 的数组
    max_val = np.max(hist_data)
    
    if max_val == 0:
        return np.zeros_like(hist_data)

    # 线性缩放
    normalized = (hist_data / max_val) * normalization
    
    # 为了防止有数据但高度不足 1 像素导致“看不见”
    # 我们可以给非零数据一个最小高度 1
    normalized[(hist_data > 0) & (normalized < 1)] = 1
    
    return normalized.astype(np.int32)

def normalize_log(hist_data, normalization=256):
    # 核心公式：y = log(count + 1)
    # +1 是为了避免 log(0)
    log_data = np.log1p(hist_data) 
    
    max_log = np.max(log_data)
    
    if max_log == 0:
        return np.zeros_like(hist_data)
        
    normalized = (log_data / max_log) * normalization
    
    return normalized.astype(np.int32)

def get_ui_ready_hist(image_16bit, downsampling=1024, normalization=256, mode='log'):
    # 1. 计算全精度直方图 (65536 bins)
    hist_raw, _ = np.histogram(image_16bit, bins=65536, range=(0, 65535))
    
    # 2. X轴降采样：将 65536 映射到 downsampling
    # 比如 65536 / 1024 = 64，即每 64 个原始 bin 取最大值或平均值
    reshaped = hist_raw.reshape(downsampling, -1)
    ui_hist_data = np.max(reshaped, axis=1) # 取最大值能更好地捕捉“溢出”倾向
    
    # 3. Y轴归一化
    if mode == 'log':
        return normalize_log(ui_hist_data, normalization)
    else:
        return normalize_linear(ui_hist_data, normalization)
```
