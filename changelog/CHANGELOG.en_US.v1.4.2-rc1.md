# v1.4.2-rc1

## New Features

- **Image Rotation**: Added clockwise and counterclockwise rotation controls for images
- Rotation angles: 0°, 90°, 180°, 270°
- Each image maintains its own rotation angle independently
- Rotation controls available in photo editing interface
- Updated CLI with `--rotate-clockwise` parameter

## Improvements

- Enhanced photo editing workflow with rotation support
- Improved batch processing to handle image rotation
