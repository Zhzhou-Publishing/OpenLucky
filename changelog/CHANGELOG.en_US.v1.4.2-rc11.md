# Changelog for v1.4.2-rc11

## New Features

- **Version Recall Protection**: Added automatic version recall detection to ensure users are using supported versions
- **Multi-language Support**: Update dialogs now support both English and Chinese (Simplified)
- **Optimized Update Checks**: Improved update check logic to reduce unnecessary API calls

## Bug Fixes

- Fixed update check logic to better handle version recall scenarios

## Improvements

- Enhanced error handling for network failures during update checks
- Improved user experience with clearer update notifications
