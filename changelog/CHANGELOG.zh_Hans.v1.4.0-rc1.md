# 更新日志

## v1.4.0-rc1 (2026-04-07)

### 新功能
- **PhotoEdit 页面布局重构**: PhotoEdit 页面的全新框架布局设计
  - 缩略图导航移至左侧，支持垂直滚动
  - 顶部固定标题栏，底部固定操作区域
  - 主图显示区域严格的高度限制
  - 支持鼠标滚轮滚动缩略图列表
  - 缩略图容器从标题栏延伸到页面底部
  - 图片容器严格限制以避免与操作区域重叠

### 改进
- **导航控制**: 更新图片导航的键盘快捷键
  - 添加上/下方向键切换图片
  - 移除左/右方向键和括号快捷键
  - 改善键盘导航体验

### Bug 修复
- 修复在预设应用和保存操作期间的窗口标题更新
- 改进不同操作期间的窗口标题同步

### 技术变更
- 重构 PhotoEdit.vue 组件结构
- 更新 CSS 布局系统以获得更好的响应性
- 增强缩略图滚动行为
- 改进图片显示定位和大小调整

### 文档
- 添加 PhotoEdit 布局重构的 PR 文档 (pr/011.photo_edit_new_frame.md)
- 添加窗口标题功能的 PR 文档 (pr/010.title_status.md)

## 安装

对于使用安装程序的 Windows 用户，只需运行 `openlucky-setup.exe` 文件并按照安装向导进行操作。

对于开发或手动安装，请参考项目文档。

---

## 升级说明

此版本包含对 PhotoEdit 页面布局的重大更改。习惯于之前布局的用户可能会注意到：
- 缩略图现在显示在左侧而不是底部
- 导航控件已移至屏幕底部
- 键盘快捷键已更改（使用上/下箭头而不是左/右箭头）
